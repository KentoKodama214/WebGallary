# セキュリティ

## 認証方式

Spring Securityによるフォームベース認証を採用しています。

## セキュリティ設定

| 項目 | 設定内容 |
|------|----------|
| パスワードハッシュ | BCrypt |
| 最大セッション数 | ユーザーあたり1セッション |
| アカウントロック | ログイン失敗3回でロック |

## アクセス制御

| リソース | アクセス制御 |
|----------|--------------|
| 静的リソース（`/css/**`, `/js/**`, `/image/**`） | 公開 |
| 写真の閲覧 | 公開（未ログインでも可能） |
| 写真の編集・アップロード | 認証必須 |
| アカウント設定 | 認証必須（本人のみ） |
